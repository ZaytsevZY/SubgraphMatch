# 文献调研子目录

此目录用于存放论文阅读过程中的中间材料，和根目录下的 `docs/literature-review.md` 配合使用。

建议文件组织如下：
- `paper-selection.md`：候选论文比较、打分和最终选择理由
- `taxonomy.md`：文献分类体系草稿
- `notes-paper-13.md`：论文 `[13]` 阅读笔记
- `notes-paper-14.md`：论文 `[14]` 阅读笔记
- `notes-paper-17.md`：论文 `[17]` 阅读笔记

建议每份阅读笔记都至少回答以下问题：
- 论文解决什么问题
- 核心算法是什么
- 有哪些过滤或加速策略
- 是否适合做课程项目复现
- 数据集是否容易获得
- 能否自然支持消融分析
