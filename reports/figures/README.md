# Report Figures

All figures in the report are generated by the Python scripts in this directory
from committed CSV data, so every result figure ships with its plotting source
(as required by the assignment).

| Figure | Script | Data | Used in |
| --- | --- | --- | --- |
| `ablation.pdf` | `plot_ablation.py` | `ablation_data.csv` | Ablation Study (Fig.\ ref `fig:ablation`) |
| `dynamic_speedup.pdf` | `plot_dynamic.py` | `dynamic_data.csv` | Optional: Dynamic Maintenance (`fig:dynamic`) |
| `homomorphism.pdf` | `plot_homomorphism.py` | `homomorphism_data.csv` | Optional: Subgraph Homomorphism (`fig:homomorphism`) |

## Regenerate

```bash
pip install matplotlib numpy
python3 reports/figures/plot_ablation.py
python3 reports/figures/plot_dynamic.py
python3 reports/figures/plot_homomorphism.py
```

Each script writes both a `.pdf` (embedded in the LaTeX report) and a `.png`
(for the presentation slides).

## Data provenance

- `ablation_data.csv`: the `|Partial delta (%)|` values from the ablation table,
  themselves produced by the GuP-lite ablation runs (`scripts/run_gup_batch.py`).
- `dynamic_data.csv`: incremental vs.\ scratch runtimes from
  `scripts/run_dynamic_batch.py` on the demo-stress graph
  (`results/raw/dyn-demo_*.json`).
- `homomorphism_data.csv`: isomorphism and non-injective homomorphism counts from
  `scripts/run_homomorphism_experiment.py` (`results/raw/hom-*.json`).
